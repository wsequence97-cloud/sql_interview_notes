# 字符串区间展开

## 1. 题目描述

给定表 `data`，表中有一个字符串字段，字段如下：

| 字段名 | 含义 |
|---|---|
| `range_str` | 用逗号分隔的数字或数字区间字符串 |

示例数据：

| range_str |
|---|
| 1-5,16,11-13,9 |

要求查询：将字符串 `1-5,16,11-13,9` 展开成 `1,2,3,4,5,16,11,12,13,9`。

> 本题口径：逗号分隔后的原始片段顺序不能改变；`1-5` 表示从 1 展开到 5，`16` 和 `9` 这种单个数字保持原值。

## 2. 期望结果

| result |
|---|
| 1,2,3,4,5,16,11,12,13,9 |

## 3. 标准答案

```sql
with data as (
    select '1-5,16,11-13,9' as range_str
),
segment_data as (
    select
        segment,
        pos + 1 as rn
    from data
    lateral view posexplode(split(range_str, ',')) t as pos, segment
),
range_data as (
    select
        segment,
        rn,
        cast(split(segment, '-')[0] as int) as start_num,
        cast(coalesce(split(segment, '-')[1], split(segment, '-')[0]) as int) as end_num
    from segment_data
),
expanded_data as (
    select
        rn,
        pos,
        start_num + pos as num
    from range_data
    lateral view posexplode(split(repeat(',', end_num - start_num), ',')) t as pos, val
),
ordered_data as (
    select
        rn,
        pos,
        num
    from expanded_data
    order by rn, pos
)
select
    concat_ws(',', collect_list(cast(num as string))) as result
from ordered_data;
```
