# 各类别销售额前三商品

## 1. 题目描述

题目来源：牛客 SQL100（中等），[官方题目](https://www.nowcoder.com/practice/3d70132f4c14442cada25fec0198e743)。公开解法依据：[牛客题解](https://www.nowcoder.com/discuss/886351696257880064)。
电商平台需要评估商品的销售绩效和利润贡献。给定商品类别表 `product_category` 和销售与利润表 `sales_and_profit`。

`product_category` 表字段如下：

| 字段名 | 含义 |
|---|---|
| `product_id` | 商品 ID |
| `category_id` | 商品所属的类别 ID |
| `product_name` | 商品名称 |

示例数据：

| product_id | category_id | product_name |
|---:|---:|---|
| 1 | 1 | 商品 A |
| 2 | 1 | 商品 B |
| 3 | 1 | 商品 C |
| 4 | 1 | 商品 D |
| 5 | 2 | 商品 1 |
| 6 | 2 | 商品 2 |
| 7 | 2 | 商品 3 |
| 8 | 2 | 商品 4 |

`sales_and_profit` 表字段如下：

| 字段名 | 含义 |
|---|---|
| `product_id` | 商品 ID，用于关联商品类别表 |
| `sales_amount` | 销售金额 |
| `cost_amount` | 成本金额 |

示例数据：

| product_id | sales_amount | cost_amount |
|---:|---:|---:|
| 1 | 500 | 50 |
| 2 | 800 | 200 |
| 3 | 8000 | 4000 |
| 4 | 700 | 600 |
| 5 | 800 | 500 |
| 6 | 1000 | 500 |
| 7 | 200 | 50 |
| 8 | 800 | 500 |

要求查询每个类别中销售金额排名前三且利润率超过 20% 的商品，输出商品 ID、商品名称、类别 ID、销售金额和利润率。利润率计算公式为 `(销售金额 - 成本金额) / 销售金额`。

结果先按类别 ID 升序排列，再按销售金额降序排列；销售金额相同时，按商品 ID 升序排列。

> 本题口径：先按类别和商品汇总销售金额及成本金额，再按销售金额降序、商品 ID 升序确定每个类别的前三名，最后从前三名中筛选利润率大于 20% 的商品。

## 3. 标准答案

```sql
WITH data AS (
    SELECT stack(
        8,
        1, 1, '商品 A', 500.00, 50.00,
        2, 1, '商品 B', 800.00, 200.00,
        3, 1, '商品 C', 8000.00, 4000.00,
        4, 1, '商品 D', 700.00, 600.00,
        5, 2, '商品 1', 800.00, 500.00,
        6, 2, '商品 2', 1000.00, 500.00,
        7, 2, '商品 3', 200.00, 50.00,
        8, 2, '商品 4', 800.00, 500.00
    ) AS (product_id, category_id, product_name, sales_amount, cost_amount)
),
product_category AS (
    SELECT
        product_id,
        category_id,
        product_name
    FROM data
),
sales_and_profit AS (
    SELECT
        product_id,
        sales_amount,
        cost_amount
    FROM data
),
product_metrics AS (
    SELECT
        p.product_id,
        p.product_name,
        p.category_id,
        sum(s.sales_amount) AS sales_amount,
        (sum(s.sales_amount) - sum(s.cost_amount)) / sum(s.sales_amount) AS profit_rate
    FROM product_category p
    INNER JOIN sales_and_profit s
        ON p.product_id = s.product_id
    GROUP BY
        p.product_id,
        p.product_name,
        p.category_id
),
ranked_products AS (
    SELECT
        product_id,
        product_name,
        category_id,
        sales_amount,
        profit_rate,
        row_number() OVER (
            PARTITION BY category_id
            ORDER BY sales_amount DESC, product_id ASC
        ) AS sales_rank
    FROM product_metrics
)
SELECT
    product_id,
    product_name,
    category_id,
    sales_amount,
    round(profit_rate, 2) AS profit_rate
FROM ranked_products
WHERE sales_rank <= 3
  AND profit_rate > 0.20
ORDER BY
    category_id ASC,
    sales_amount DESC,
    product_id ASC;
```
