# 每月播放量前三的周杰伦歌曲

## 1. 题目描述

题目来源：[牛客 SQL40 每个月 Top3 的周杰伦歌曲](https://www.nowcoder.com/practice/4ab6d198ea8447fe9b6a1cad1f671503)（较难）。

给定听歌流水表 `play_log`，字段如下：

| 字段名 | 含义 |
|---|---|
| `fdate` | 播放日期 |
| `user_id` | 用户 ID |
| `song_id` | 歌曲 ID |

示例数据：

| fdate | user_id | song_id |
|---|---:|---:|
| 2022-01-08 | 10000 | 0 |
| 2022-01-16 | 10000 | 0 |
| 2022-01-20 | 10000 | 0 |
| 2022-01-25 | 10000 | 0 |
| 2022-01-02 | 10000 | 1 |
| 2022-01-12 | 10000 | 1 |
| 2022-01-13 | 10000 | 1 |
| 2022-01-14 | 10000 | 1 |
| 2022-01-10 | 10000 | 2 |
| 2022-01-11 | 10000 | 3 |
| 2022-01-16 | 10000 | 3 |
| 2022-01-11 | 10000 | 4 |
| 2022-01-27 | 10000 | 4 |
| 2022-02-05 | 10000 | 0 |
| 2022-02-19 | 10000 | 0 |
| 2022-02-07 | 10000 | 1 |
| 2022-02-27 | 10000 | 2 |
| 2022-02-25 | 10000 | 3 |
| 2022-02-03 | 10000 | 4 |
| 2022-02-16 | 10000 | 4 |

给定歌曲表 `song_info`，字段如下：

| 字段名 | 含义 |
|---|---|
| `song_id` | 歌曲 ID |
| `song_name` | 歌曲名称 |
| `singer_name` | 歌手名称 |

示例数据：

| song_id | song_name | singer_name |
|---:|---|---|
| 0 | 明明就 | 周杰伦 |
| 1 | 说好的幸福呢 | 周杰伦 |
| 2 | 江南 | 林俊杰 |
| 3 | 大笨钟 | 周杰伦 |
| 4 | 黑键 | 林俊杰 |

给定用户表 `user_info`，字段如下：

| 字段名 | 含义 |
|---|---|
| `user_id` | 用户 ID |
| `age` | 用户年龄 |

示例数据：

| user_id | age |
|---:|---:|
| 10000 | 18 |

要求查询：从听歌流水中找出 18 至 25 岁用户在 2022 年每个月播放次数排名前三的周杰伦歌曲，输出月份、排名、歌曲名称和播放次数。

> 本题口径：同一月份内按播放次数降序排名；播放次数相同时，按 `song_id` 升序确定先后顺序。

## 3. 标准答案

以下答案根据[公开题解](https://www.nowcoder.com/discuss/893584208243740672)的思路改写为 Hive SQL。

```sql
WITH data AS (
    SELECT stack(
        20,
        '2022-01-08', 10000, 0,
        '2022-01-16', 10000, 0,
        '2022-01-20', 10000, 0,
        '2022-01-25', 10000, 0,
        '2022-01-02', 10000, 1,
        '2022-01-12', 10000, 1,
        '2022-01-13', 10000, 1,
        '2022-01-14', 10000, 1,
        '2022-01-10', 10000, 2,
        '2022-01-11', 10000, 3,
        '2022-01-16', 10000, 3,
        '2022-01-11', 10000, 4,
        '2022-01-27', 10000, 4,
        '2022-02-05', 10000, 0,
        '2022-02-19', 10000, 0,
        '2022-02-07', 10000, 1,
        '2022-02-27', 10000, 2,
        '2022-02-25', 10000, 3,
        '2022-02-03', 10000, 4,
        '2022-02-16', 10000, 4
    ) AS (fdate, user_id, song_id)
),
song_info AS (
    SELECT stack(
        5,
        0, '明明就', '周杰伦',
        1, '说好的幸福呢', '周杰伦',
        2, '江南', '林俊杰',
        3, '大笨钟', '周杰伦',
        4, '黑键', '林俊杰'
    ) AS (song_id, song_name, singer_name)
),
user_info AS (
    SELECT stack(
        1,
        10000, 18
    ) AS (user_id, age)
),
filtered_plays AS (
    SELECT
        month(to_date(p.fdate)) AS month,
        p.song_id,
        s.song_name
    FROM data p
    JOIN song_info s
        ON p.song_id = s.song_id
    JOIN user_info u
        ON p.user_id = u.user_id
    WHERE u.age BETWEEN 18 AND 25
      AND year(to_date(p.fdate)) = 2022
      AND s.singer_name = '周杰伦'
),
song_play_counts AS (
    SELECT
        month,
        song_id,
        song_name,
        count(*) AS play_pv
    FROM filtered_plays
    GROUP BY month, song_id, song_name
),
ranked_songs AS (
    SELECT
        month,
        song_id,
        song_name,
        play_pv,
        row_number() OVER (
            PARTITION BY month
            ORDER BY play_pv DESC, song_id ASC
        ) AS ranking
    FROM song_play_counts
)
SELECT
    month,
    ranking,
    song_name,
    play_pv
FROM ranked_songs
WHERE ranking <= 3
ORDER BY month, ranking;
```
