# 查询股票波峰波谷

## 1. 题目描述

给定股票价格表 `test`，每一行表示某只股票在某一天的价格。

| 字段名 | 含义 |
|---|---|
| `id` | 股票 ID |
| `ds` | 日期 |
| `price` | 当日价格 |

示例数据：

| id | ds | price |
|---:|---|---:|
| 1 | 2019-01-01 | 10001 |
| 1 | 2019-01-03 | 1001 |
| 1 | 2019-01-02 | 10002 |
| 1 | 2019-01-04 | 999 |
| 1 | 2019-01-05 | 1002 |
| 1 | 2019-01-06 | 1003 |
| 1 | 2019-01-07 | 1004 |
| 1 | 2019-01-08 | 998 |
| 1 | 2019-01-09 | 997 |
| 1 | 2019-01-10 | 996 |
| 2 | 2019-01-01 | 10001 |
| 2 | 2019-01-02 | 10002 |
| 2 | 2019-01-03 | 10003 |
| 2 | 2019-01-04 | 10002 |
| 2 | 2019-01-05 | 1002 |
| 2 | 2019-01-06 | 1003 |
| 2 | 2019-01-07 | 1004 |
| 2 | 2019-01-08 | 998 |
| 2 | 2019-01-09 | 997 |
| 2 | 2019-01-10 | 996 |

要求找出每只股票的波峰和波谷。

> 本题口径：波峰指当天价格大于前一天和后一天；波谷指当天价格小于前一天和后一天。每只股票按 `ds` 日期排序后比较相邻记录，第一天和最后一天因为缺少前后比较对象，不判断为波峰或波谷。

## 2. 期望结果

| id | ds | price | price_type |
|---:|---|---:|---|
| 1 | 2019-01-02 | 10002 | 波峰 |
| 1 | 2019-01-04 | 999 | 波谷 |
| 1 | 2019-01-07 | 1004 | 波峰 |
| 2 | 2019-01-03 | 10003 | 波峰 |
| 2 | 2019-01-05 | 1002 | 波谷 |
| 2 | 2019-01-07 | 1004 | 波峰 |

## 3. 标准答案

```sql
with data as (
    select 1 as id, '2019-01-01' as ds, 10001 as price union all
    select 1 as id, '2019-01-03' as ds, 1001 as price union all
    select 1 as id, '2019-01-02' as ds, 10002 as price union all
    select 1 as id, '2019-01-04' as ds, 999 as price union all
    select 1 as id, '2019-01-05' as ds, 1002 as price union all
    select 1 as id, '2019-01-06' as ds, 1003 as price union all
    select 1 as id, '2019-01-07' as ds, 1004 as price union all
    select 1 as id, '2019-01-08' as ds, 998 as price union all
    select 1 as id, '2019-01-09' as ds, 997 as price union all
    select 1 as id, '2019-01-10' as ds, 996 as price union all
    select 2 as id, '2019-01-01' as ds, 10001 as price union all
    select 2 as id, '2019-01-02' as ds, 10002 as price union all
    select 2 as id, '2019-01-03' as ds, 10003 as price union all
    select 2 as id, '2019-01-04' as ds, 10002 as price union all
    select 2 as id, '2019-01-05' as ds, 1002 as price union all
    select 2 as id, '2019-01-06' as ds, 1003 as price union all
    select 2 as id, '2019-01-07' as ds, 1004 as price union all
    select 2 as id, '2019-01-08' as ds, 998 as price union all
    select 2 as id, '2019-01-09' as ds, 997 as price union all
    select 2 as id, '2019-01-10' as ds, 996 as price
),
base as (
    select
        id,
        to_date(ds) as ds,
        price
    from data
),
ordered_data as (
    select
        id,
        ds,
        price,
        lag(price, 1) over(partition by id order by ds) as lag_price,
        lead(price, 1) over(partition by id order by ds) as lead_price
    from base
),
marked_data as (
    select
        id,
        ds,
        price,
        case
            when price > lag_price and price > lead_price then '波峰'
            when price < lag_price and price < lead_price then '波谷'
        end as price_type
    from ordered_data
)
select
    id,
    date_format(ds, 'yyyy-MM-dd') as ds,
    price,
    price_type
from marked_data
where price_type is not null
order by id, ds;
```

    end price_type

    base

## 6. 我的复盘要点

- 【本题坑点】样例里 `id = 1` 的日期不是按顺序给出的，所以比较前后价格时必须 `partition by id order by ds`。
- 【修正记法】波峰就是当前价格同时大于前一天和后一天；波谷就是当前价格同时小于前一天和后一天。
- 【我的卡点】我的判断逻辑基本对，但如果题目要求“找出波峰波谷”，最终还要过滤 `price_type is not null`，否则普通日期也会带着空类型输出。
- 【本题坑点】第一行缺前一天、最后一行缺后一天，`lag/lead` 会返回空，这两类边界行自然不能判断为波峰或波谷。
