# 按学生恢复班级和分数列表

## 1. 题目描述

给定学生成绩明细表 `data`，字段如下：

| 字段名 | 含义 |
|---|---|
| `class_name` | 班级 |
| `student` | 学生姓名 |
| `score` | 分数 |

示例数据：

| class_name | student | score |
|---|---|---:|
| a | yuxing | 100 |
| c | yuxing1 | 60 |
| b | yuxing | 80 |
| b | yuxing1 | 80 |
| c | yuxing | 90 |

要求查询：按 `student` 聚合，把同一个学生对应的班级和分数恢复成列表。

> 本题口径：图片底部结果是数组形式，因此标准答案输出 `collect_list` 聚合后的数组。如果业务要求输出逗号拼接字符串，则在 `collect_list` 外层再套 `concat_ws(',', ...)`。

## 2. 期望结果

| student | class_list | score_list |
|---|---|---|
| yuxing | ["a","b","c"] | ["100","80","90"] |
| yuxing1 | ["c","b"] | ["60","80"] |

## 3. 标准答案

```sql
with data as (
    select 'a' as class_name, 'yuxing' as student, 100 as score union all
    select 'c' as class_name, 'yuxing1' as student, 60 as score union all
    select 'b' as class_name, 'yuxing' as student, 80 as score union all
    select 'b' as class_name, 'yuxing1' as student, 80 as score union all
    select 'c' as class_name, 'yuxing' as student, 90 as score
),
base as (
    select
        class_name,
        student,
        cast(score as string) as score
    from data
)
select
    student,
    collect_list(class_name) as class_list,
    collect_list(score) as score_list
from base
group by student
order by student;
```

## 5. 我的解答正确版本

```sql
with data as (
    select 'a' as class_name, 'yuxing' as student, 100 as score union all
    select 'c' as class_name, 'yuxing1' as student, 60 as score union all
    select 'b' as class_name, 'yuxing' as student, 80 as score union all
    select 'b' as class_name, 'yuxing1' as student, 80 as score union all
    select 'c' as class_name, 'yuxing' as student, 90 as score
)
select 
    student,
    collect_list(class_name) as class_list,
    collect_list(cast(score as string)) as score_list
from 
    data 
group by 
    student
order by 
    student;
```

你的思路是对的：本题就是按 `student` 分组后，用 `collect_list` 把多行班级、分数收回到数组里。这里主要补充两点：给输出列加上明确列名，且如果期望结果展示为字符串数组，可以把 `score` 先转成 `string`。

## 6. 我的复盘要点

- 【我的卡点】`collect_list(col)` 是聚合函数，要和 `group by` 配合使用：`group by student` 后，`collect_list(class_name)` 会把同一个学生的多行班级值收集成一个数组，例如 `["a","b","c"]`。它解决的是“多行收缩成一行”的问题，和 `explode` 的“数组展开成多行”正好相反。
- 【修正记法】`collect_list` 不去重，原表有几个值就收几个值；如果同一个学生有重复班级，数组里也会保留重复值。若业务要求去重，才考虑 `collect_set(col)`，但 `collect_set` 同样不保证顺序。
- 【本题坑点】`collect_list(score)` 收出来的是数字数组，`collect_list(cast(score as string))` 收出来的是字符串数组；如果后续要用 `concat_ws(',', collect_list(...))` 拼字符串，数组元素最好先转成 `string`。
- 【下次先想】先判断最终输出形态：要数组就写 `collect_list(col)`；要逗号分隔字符串就写 `concat_ws(',', collect_list(cast(col as string)))`。图片里底部结果是 `["100","80","90"]` 这种数组，所以标准答案保留 `collect_list`。
- 【本题坑点】`collect_list` 默认不保证稳定顺序，本题结果看起来是 `a,b,c`，但 Hive 分布式执行时不能只靠原始插入顺序。若面试题要求顺序严格稳定，需要先明确排序字段，或者用结构体、排序后再收集的方式保证班级列表和分数列表一一对应。
