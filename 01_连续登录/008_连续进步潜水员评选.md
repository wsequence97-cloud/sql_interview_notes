## 1. 题目描述

- 牛客题号：SQL150
- 题目：25年携程-海岛潜水基地「连续进步」潜水员评选
- 难度：较难
- 原题链接：https://www.nowcoder.com/practice/05994083a6184554947aac96c568baaf
- 公开题解依据：https://www.nowcoder.com/discuss/892148975673831424

某海岛潜水基地每年 3 月至 9 月是旺季。基地总教头需要找出 2025 年旺季中每个月潜次数和平均潜深都持续不降的潜水员，并为每个符合条件的三个月窗口单独输出一行。

| 字段名 | 类型 | 含义 |

| `nationality` | VARCHAR(32) | 国籍 |

> 本题口径：M0、M1、M2 必须是三个连续自然月且每月都有潜次记录；潜次数满足 `m0_dives <= m1_dives <= m2_dives`；平均潜深满足 `m0_avg_depth <= m1_avg_depth <= m2_avg_depth`；窗口起始月还必须满足 `m0_dives >= 2`。月平均潜深四舍五入保留 2 位。结果严格按 `total_window_dives` 降序、`window_m0` 升序、`diver_id` 升序排列。

monthly_stats AS (
    SELECT
        diver_id,
        to_date(concat(date_format(dive_date, 'yyyy-MM'), '-01')) AS month_start,
        count(*) AS month_dives,
        CAST(round(avg(depth_m), 2) AS DECIMAL(10, 2)) AS avg_depth

            ORDER BY month_start

            ORDER BY month_start

            PARTITION BY diver_id
            ORDER BY month_start

    SELECT
        diver_id,
        window_m0,
        window_m1,
        window_m2,
        m0_dives,
        m1_dives,
        m2_dives,
        m0_avg_depth,
        m1_avg_depth,
        m2_avg_depth,

      AND window_m2 = add_months(window_m0, 2)
      AND m0_dives >= 2
      AND m0_dives <= m1_dives
      AND m1_dives <= m2_dives
      AND m0_avg_depth <= m1_avg_depth
      AND m1_avg_depth <= m2_avg_depth

    date_format(qw.window_m2, 'yyyy-MM') AS window_m2,

    qw.m1_dives,
    qw.m2_dives,

    qw.m1_avg_depth,
    qw.m2_avg_depth,
    qw.total_window_dives
FROM qualified_windows qw
INNER JOIN t_diver d
    ON qw.diver_id = d.diver_id
ORDER BY
    qw.total_window_dives DESC,
    qw.window_m0 ASC,
    qw.diver_id ASC;
