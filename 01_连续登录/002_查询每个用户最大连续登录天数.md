# 查询每个用户最大连续登录天数

## 1. 题目描述

给定用户登录记录表 `data`，每一行表示某个用户的一次登录时间。

| 字段名 | 含义 |
|---|---|
| `id` | 用户 ID |
| `date` | 登录时间字符串 |

示例数据：

| id | date |
|---:|---|
| 1 | 2019-01-01 00:00:00 |
| 1 | 2019-01-02 00:00:00 |
| 1 | 2019-01-03 00:00:00 |
| 1 | 2019-01-05 00:00:00 |
| 2 | 2019-02-01 00:00:00 |
| 2 | 2019-02-02 00:00:00 |
| 2 | 2019-02-04 00:00:00 |
| 3 | 2019-03-04 00:00:00 |
| 3 | 2019-03-05 00:00:00 |
| 3 | 2019-03-06 00:00:00 |
| 3 | 2019-03-07 00:00:00 |

要求查询每个用户的最大连续登录天数，输出用户 ID 和该用户历史上最长的一段连续登录天数 `max_day_cnt`。

> 本题口径：同一用户同一天可能有多次登录，计算前需要按天去重。

## 2. 期望结果

| id | max_day_cnt |
|---:|---:|
| 1 | 3 |
| 2 | 2 |
| 3 | 4 |

## 3. 标准答案

```sql
with data as (
    select 1 as id, '2019-01-01 00:00:00' as `date` union all
    select 1 as id, '2019-01-02 00:00:00' as `date` union all
    select 1 as id, '2019-01-03 00:00:00' as `date` union all
    select 1 as id, '2019-01-05 00:00:00' as `date` union all
    select 2 as id, '2019-02-01 00:00:00' as `date` union all
    select 2 as id, '2019-02-02 00:00:00' as `date` union all
    select 2 as id, '2019-02-04 00:00:00' as `date` union all
    select 3 as id, '2019-03-04 00:00:00' as `date` union all
    select 3 as id, '2019-03-05 00:00:00' as `date` union all
    select 3 as id, '2019-03-06 00:00:00' as `date` union all
    select 3 as id, '2019-03-07 00:00:00' as `date`
),
login_distinct as (
    select distinct
        id,
        to_date(`date`) as login_date
    from data
),
login_ranked as (
    select
        id,
        login_date,
        row_number() over(partition by id order by login_date) as rn
    from login_distinct
),
login_grouped as (
    select
        id,
        login_date,
        date_sub(login_date, cast(rn as int)) as date1
    from login_ranked
),
login_day_cnt as (
    select
        id,
        date1,
        count(*) as day_cnt
    from login_grouped
    group by id, date1
)
select
    id,
    max(day_cnt) as max_day_cnt
from login_day_cnt
group by id
order by id;
```

```sql
    select 1 as id, '2019-01-02 00:00:00' as date union all

    select 2 as id, '2019-02-01 00:00:00' as date union all

    select 2 as id, '2019-02-04 00:00:00' as date union all
    select 3 as id, '2019-03-04 00:00:00' as date union all
    select 3 as id, '2019-03-05 00:00:00' as date union all
    select 3 as id, '2019-03-06 00:00:00' as date union all
    select 3 as id, '2019-03-07 00:00:00' as date
)

select
    id,
    max(date_cnt)
from(
    select
        id,
        count(date2) date_cnt
    from(
        select
            id,
            DATE_ADD(date, -row_number() over(partition by id order by date)) date2
        from
            data
    )t2
    group by
        id,date2
)t3
group by
    id
```

```sql
with data as (
    select 1 as id, '2019-01-01 00:00:00' as `date` union all
    select 1 as id, '2019-01-02 00:00:00' as `date` union all
    select 1 as id, '2019-01-03 00:00:00' as `date` union all
    select 1 as id, '2019-01-05 00:00:00' as `date` union all
    select 2 as id, '2019-02-01 00:00:00' as `date` union all
    select 2 as id, '2019-02-02 00:00:00' as `date` union all
    select 2 as id, '2019-02-04 00:00:00' as `date` union all
    select 3 as id, '2019-03-04 00:00:00' as `date` union all
    select 3 as id, '2019-03-05 00:00:00' as `date` union all
    select 3 as id, '2019-03-06 00:00:00' as `date` union all
    select 3 as id, '2019-03-07 00:00:00' as `date`
),
login_distinct as (
    select distinct
        id,
        to_date(`date`) as login_date
    from data
),
login_grouped as (
    select
        id,
        date_sub(
            login_date,
            cast(row_number() over(partition by id order by login_date) as int)
        ) as date1
    from login_distinct
),
login_day_cnt as (
    select
        id,
        date1,
        count(*) as date_cnt
    from login_grouped
    group by id, date1

    id,
    max(date_cnt) as max_day_cnt
from login_day_cnt
group by id
order by id;
```

## 6. 我的复盘要点

- 【我的卡点】这题不是直接对每个用户 `count`，而是要先拆出每一段连续登录，再对每个用户取最大段长。
- 【修正记法】最大连续登录天数的顺序是：按天去重 -> 日期减行号分段 -> 每段 `count(*)` -> 每个用户 `max()`。
- 【本题坑点】如果同一用户同一天有多次登录，必须先按 `id + login_date` 去重，否则 `row_number()` 会把同一天重复记录也算进连续段。
