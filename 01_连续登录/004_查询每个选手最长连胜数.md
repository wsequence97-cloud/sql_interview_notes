# 查询每个选手最长连胜数

## 1. 题目描述

给定比赛结果表 `matches`，每一行表示某位选手在某一天的一场比赛结果。

| 字段名 | 含义 |
|---|---|
| `player_id` | 参赛选手 ID |
| `match_day` | 比赛日期 |
| `result` | 比赛结果，取值为 `Win`、`Draw`、`Lose` |

示例数据：

| player_id | match_day | result |
|---:|---|---|
| 1 | 2023-01-01 | Win |
| 1 | 2023-01-02 | Win |
| 1 | 2023-01-03 | Draw |
| 1 | 2023-01-04 | Win |
| 1 | 2023-01-05 | Win |
| 1 | 2023-01-06 | Win |
| 1 | 2023-01-07 | Lose |
| 2 | 2023-01-01 | Lose |
| 2 | 2023-01-02 | Draw |
| 2 | 2023-01-03 | Win |
| 2 | 2023-01-04 | Win |
| 3 | 2023-01-01 | Draw |
| 3 | 2023-01-02 | Lose |

要求计算每个参赛选手最多的连胜数。选手的连胜数是指连续获胜的次数，且没有被平局或输球中断。

> 本题口径：`Draw` 和 `Lose` 都会打断连胜；如果某个选手从未赢过比赛，最长连胜数记为 `0`。截图中没有提供完整样例数据和期望结果，当前样例数据为根据题意构造。

## 2. 期望结果

| player_id | longest_streak |
|---:|---:|
| 1 | 3 |
| 2 | 2 |
| 3 | 0 |

## 3. 标准答案

```sql
with data as (
    select 1 as player_id, '2023-01-01' as match_day, 'Win' as result union all
    select 1 as player_id, '2023-01-02' as match_day, 'Win' as result union all
    select 1 as player_id, '2023-01-03' as match_day, 'Draw' as result union all
    select 1 as player_id, '2023-01-04' as match_day, 'Win' as result union all
    select 1 as player_id, '2023-01-05' as match_day, 'Win' as result union all
    select 1 as player_id, '2023-01-06' as match_day, 'Win' as result union all
    select 1 as player_id, '2023-01-07' as match_day, 'Lose' as result union all
    select 2 as player_id, '2023-01-01' as match_day, 'Lose' as result union all
    select 2 as player_id, '2023-01-02' as match_day, 'Draw' as result union all
    select 2 as player_id, '2023-01-03' as match_day, 'Win' as result union all
    select 2 as player_id, '2023-01-04' as match_day, 'Win' as result union all
    select 3 as player_id, '2023-01-01' as match_day, 'Draw' as result union all
    select 3 as player_id, '2023-01-02' as match_day, 'Lose' as result
),
base as (
    select
        player_id,
        to_date(match_day) as match_day,
        result
    from data
),
players as (
    select distinct
        player_id
    from base
),
marked_data as (
    select
        player_id,
        match_day,
        result,
        sum(case when result <> 'Win' then 1 else 0 end) over(
            partition by player_id
            order by match_day
            rows between unbounded preceding and current row
        ) as break_group
    from base
),
win_streaks as (
    select
        player_id,
        break_group,
        count(*) as streak_cnt
    from marked_data
    where result = 'Win'
    group by player_id, break_group
),
max_streak as (
    select
        player_id,
        max(streak_cnt) as longest_streak
    from win_streaks
    group by player_id
)
select
    p.player_id,
    coalesce(m.longest_streak, 0) as longest_streak
from players p
left join max_streak m
    on p.player_id = m.player_id
order by p.player_id;
```

marked_data as (
    select 
        player_id,
        match_day,
        result,
        case 
            when order_result is null then 1
            when order_result <> result then 1
            else 0
        end mark_result
    from 
        ordered_data
),
grouped_data as (
    select 
        player_id,
        match_day,
        result,
        sum(mark_result) over(
            partition by player_id
            order by match_day
            rows between unbounded preceding and current row
        ) is_new_group
    from 
        marked_data
),
merged_data as (
    select 
        player_id,
        result,
        count(*) con_result
    from 
        grouped_data
    group by 
        player_id, is_new_group, result
    having 
        result = 'win'
)
select 
    player_id,
    max(con_result) 
from    
    merged_data
group by   
    player_id

    select 
        player_id,
        match_day,
        result,
        sum(is_new_group) over(
            partition by player_id
            order by match_day

        ) as group_id
    from marked_data
),
win_streaks as (
    select 
        player_id,
        group_id,
        count(*) as streak_cnt
    from grouped_data
    where result = 'Win'
    group by player_id, group_id

        player_id,

## 6. 我的复盘要点

- 【错误原因】我原 SQL 里写了 `result = 'win'`，但样例数据是 `'Win'`；Hive 字符串大小写敏感，会导致胜场匹配不到。
- 【本题坑点】`where result = 'Win'` 不能提前过滤，必须先保留 `Draw/Lose` 生成打断分组，否则连胜段会被错误连接。
- 【修正记法】连胜题可以用“累计非 Win 次数”做分组编号：遇到 `Draw/Lose` 分组编号加 1，连续的 `Win` 留在同一组。
- 【本题坑点】如果题目要求每个选手都输出，从未赢过的选手也要返回 0，需要先保留选手全集再左连接最大连胜结果。
