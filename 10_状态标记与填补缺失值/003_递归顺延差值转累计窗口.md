# 递归顺延差值转累计窗口

## 1. 题目描述

给定项目金额表 `data`，字段如下：

| 字段名 | 含义 |
|---|---|
| `t_id` | 项目 ID |
| `tt_id` | 子项目 ID |
| `money` | 当前金额 |
| `cnt` | 变动总金额 |

示例数据：

| t_id | tt_id | money | cnt |
|---|---|---:|---:|
| A | a1 | 10 | 60 |
| A | a2 | 20 | 60 |
| A | a3 | 40 | 60 |
| B | b1 | 5 | 15 |
| B | b2 | 1 | 15 |
| B | b3 | 10 | 15 |

要求查询：每个项目内，把 `cnt` 均摊到子项目行上。每行用当前金额减去当行应分摊金额；如果不足扣成负数，则当前行结果记为 0，不足部分顺延到下一行继续扣减。

> 本题口径：这题看起来像递归，实际可以用累计窗口转换。当前行结果等价于 `截至当前行累计 money - 截至当前行累计均摊金额`，若该差值小于 0 则记为 0。

## 2. 期望结果

| t_id | tt_id | difs | cnt |
|---|---|---:|---:|
| A | a1 | 0 | 60 |
| A | a2 | 0 | 60 |
| A | a3 | 10 | 60 |
| B | b1 | 0 | 15 |
| B | b2 | 0 | 15 |
| B | b3 | 1 | 15 |

## 3. 标准答案

```sql
with data as (
    select 'A' as t_id, 'a1' as tt_id, 10 as money, 60 as cnt union all
    select 'A' as t_id, 'a2' as tt_id, 20 as money, 60 as cnt union all
    select 'A' as t_id, 'a3' as tt_id, 40 as money, 60 as cnt union all
    select 'B' as t_id, 'b1' as tt_id, 5 as money, 15 as cnt union all
    select 'B' as t_id, 'b2' as tt_id, 1 as money, 15 as cnt union all
    select 'B' as t_id, 'b3' as tt_id, 10 as money, 15 as cnt
),
base as (
    select
        t_id,
        tt_id,
        money,
        cnt,
        sum(money) over (
            partition by t_id
            order by tt_id
            rows between unbounded preceding and current row
        ) as sum_money,
        avg(cnt) over (partition by t_id) / 3 as avg_money
    from data
),
diff_data as (
    select
        t_id,
        tt_id,
        cnt,
        sum_money - sum(avg_money) over (
            partition by t_id
            order by tt_id
            rows between unbounded preceding and current row
        ) as diff
    from base
)
select
    t_id,
    tt_id,
    case
        when diff < 0 then 0
        else diff
    end as difs,
    cnt
from diff_data
order by t_id, tt_id;
```

## 4. 我的解答

```sql
with data as (
    select 'A' as t_id, 'a1' as tt_id, 10 as money, 60 as cnt union all
    select 'A' as t_id, 'a2' as tt_id, 20 as money, 60 as cnt union all
    select 'A' as t_id, 'a3' as tt_id, 40 as money, 60 as cnt union all
    select 'B' as t_id, 'b1' as tt_id, 5 as money, 15 as cnt union all
    select 'B' as t_id, 'b2' as tt_id, 1 as money, 15 as cnt union all
    select 'B' as t_id, 'b3' as tt_id, 10 as money, 15 as cnt
),
ordered_data as (
    select
        t_id,
        tt_id,
        money,
        cnt,
        cnt / (count(*) over(partition by t_id)) avg_cnt
    from
        data
),
diff_data as (
    select
        t_id,
        tt_id,
        sum(money) over(partition by t_id order by tt_id) - sum(avg_cnt) over(partition by t_id) diff_money,
        cnt
    from
        ordered_data
)
select
    t_id,
    tt_id,
    case
        when diff_money < 0 then 0
        else diff_money
    end diff_money_0,
    cnt
from
    diff_data;
```

## 5. 我的解答正确版本

```sql
with data as (
    select 'A' as t_id, 'a1' as tt_id, 10 as money, 60 as cnt union all
    select 'A' as t_id, 'a2' as tt_id, 20 as money, 60 as cnt union all
    select 'A' as t_id, 'a3' as tt_id, 40 as money, 60 as cnt union all
    select 'B' as t_id, 'b1' as tt_id, 5 as money, 15 as cnt union all
    select 'B' as t_id, 'b2' as tt_id, 1 as money, 15 as cnt union all
    select 'B' as t_id, 'b3' as tt_id, 10 as money, 15 as cnt
),
ordered_data as (
    select
        t_id,
        tt_id,
        money,
        cnt,
        cnt / count(*) over(partition by t_id) as avg_cnt
    from data
),
diff_data as (
    select
        t_id,
        tt_id,
        sum(money) over(
            partition by t_id
            order by tt_id
            rows between unbounded preceding and current row
        ) - sum(avg_cnt) over(
            partition by t_id
            order by tt_id
            rows between unbounded preceding and current row
        ) as diff_money,
        cnt
    from ordered_data
)
select
    t_id,
    tt_id,
    case
        when diff_money < 0 then 0
        else diff_money
    end as diff_money_0,
    cnt
from diff_data
order by t_id, tt_id;
```

## 6. 我的复盘要点

- 【下次先想】这题表面像“上一行不足顺延到下一行”的递归题，但核心可以转成累计窗口：累计当前金额减累计应扣金额。
- 【修正记法】你的 `cnt / count(*) over(partition by t_id)` 已经算出了每行均摊额；后面也要对 `avg_cnt` 做“截至当前行”的累计，而不是直接减全组 `sum(avg_cnt)`。
- 【本题坑点】`sum(avg_cnt) over(partition by t_id)` 表示全项目应扣总额，每一行都会减同一个总额；这里需要的是 `partition by t_id order by tt_id rows between unbounded preceding and current row`。
- 【本题坑点】如果只看单行 `money - avg_cnt`，就无法体现上一行不足顺延到下一行；必须用 `累计money - 累计avg_cnt` 才能把前面不足的部分带下来。
- 【本题坑点】负数结果要转成 0，但负数背后的不足部分不能丢；它已经体现在后续行的累计差值里。
