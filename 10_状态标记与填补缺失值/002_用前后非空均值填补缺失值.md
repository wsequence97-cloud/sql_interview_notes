# 用前后非空均值填补缺失值

## 1. 题目描述

给定品种价格表 `data`，字段如下：

| 字段名 | 含义 |
|---|---|
| `breed` | 品种 |
| `dt` | 日期 |
| `value` | 当日数值，可能为空 |

|---|---|---:|

| B | 2023-03-01 | 4310 |
| B | 2023-03-02 | 4280 |
| B | 2023-03-03 | 4470 |

| C | 2022-09-30 | 4470 |
| C | 2022-10-01 | null |
| C | 2022-10-02 | null |
| C | 2022-10-03 | null |
| C | 2022-10-04 | null |
| C | 2022-10-05 | null |
| C | 2022-10-06 | null |
| C | 2022-10-07 | null |
| C | 2022-10-08 | 4480 |
| C | 2022-10-09 | 4470 |
| C | 2022-10-10 | 4520 |

要求查询：按 `breed` 和日期顺序填补中间缺失值。若某段连续空值位于前后两个非空值之间，则该段所有空值填充为前后最近非空值的平均值。

> 本题口径：只处理“中间空缺值”，即空值段前后都存在非空值；原本非空的 `value` 保持不变。

## 3. 标准答案

```sql
with data as (
    select 'A' as breed, '2023-03-01' as dt, 4230 as value union all
    select 'A' as breed, '2023-03-02' as dt, 4470 as value union all
    select 'A' as breed, '2023-03-03' as dt, 4520 as value union all
    select 'A' as breed, '2023-03-04' as dt, null as value union all
    select 'A' as breed, '2023-03-05' as dt, null as value union all
    select 'A' as breed, '2023-03-06' as dt, 4430 as value union all
    select 'B' as breed, '2023-03-01' as dt, 4310 as value union all
    select 'B' as breed, '2023-03-02' as dt, 4280 as value union all
    select 'B' as breed, '2023-03-03' as dt, 4470 as value union all
    select 'B' as breed, '2023-03-04' as dt, null as value union all
    select 'B' as breed, '2023-03-05' as dt, null as value union all
    select 'B' as breed, '2023-03-06' as dt, 4310 as value union all
    select 'C' as breed, '2022-09-27' as dt, 4280 as value union all
    select 'C' as breed, '2022-09-28' as dt, 4470 as value union all
    select 'C' as breed, '2022-09-29' as dt, 4280 as value union all
    select 'C' as breed, '2022-09-30' as dt, 4470 as value union all
    select 'C' as breed, '2022-10-01' as dt, null as value union all
    select 'C' as breed, '2022-10-02' as dt, null as value union all
    select 'C' as breed, '2022-10-03' as dt, null as value union all
    select 'C' as breed, '2022-10-04' as dt, null as value union all
    select 'C' as breed, '2022-10-05' as dt, null as value union all
    select 'C' as breed, '2022-10-06' as dt, null as value union all
    select 'C' as breed, '2022-10-07' as dt, null as value union all
    select 'C' as breed, '2022-10-08' as dt, 4480 as value union all
    select 'C' as breed, '2022-10-09' as dt, 4470 as value union all
    select 'C' as breed, '2022-10-10' as dt, 4520 as value
),
base as (
    select
        breed,
        to_date(dt) as dt,
        value
    from data
),
marked_data as (
    select
        breed,
        dt,
        value,
        last_value(value, true) over (
            partition by breed
            order by dt
            rows between unbounded preceding and current row
        ) as prev_value,
        first_value(value, true) over (
            partition by breed
            order by dt
            rows between current row and unbounded following
        ) as next_value
    from base
)
select
    breed,
    date_format(dt, 'yyyy-MM-dd') as dt,
    cast(
        coalesce(
            value,
            (prev_value + next_value) / 2
        ) as int
    ) as filled_value
from marked_data
order by breed, dt;
```

## 6. 我的复盘要点

- 【我的卡点】我的思路已经抓到了“前一个值 + 后一个值再除以 2”，但窗口必须 `partition by breed`，否则 A、B、C 不同品种会被混在同一条时间线上。
- 【修正记法】`last_value(value, true)` 向前找最近非空值，`first_value(value, true)` 向后找最近非空值；第二个参数 `true` 表示忽略空值。
- 【本题坑点】普通 `last_value(value)` / `first_value(value)` 不会跳过 null，连续空值时会拿到 null，导致 `(lag_value + lead_value) / 2` 也变成 null。
- 【本题坑点】窗口边界要覆盖当前行：前值窗口写到 `current row`，后值窗口从 `current row` 开始；配合忽略空值，当前行非空时也能保留自身。
- 【本题坑点】连续多个空值填的是同一组前后非空值的均值，例如 C 的 2022-10-01 到 2022-10-07 都是 `(4470 + 4480) / 2 = 4475`，不是逐行滚动平均。
