# SQL 面试题笔记库

## 1. 题库说明

本项目用于整理用户上传、练习和复盘过的 SQL 面试题。每道题都以独立 Markdown 文件保存，并统一使用 Hive SQL CTE 完整版作为标准答案。

每道题默认只保留 6 个部分：

1. 题目描述
2. 期望结果
3. 标准答案
4. 我的解答
5. 我的解答正确版本
6. 我的复盘要点

## 4. 个人复盘方法

1. 先看题目描述和期望结果，自己写一版 SQL。
2. 对比“标准答案”，检查 Hive CTE 分层是否清晰。
3. 阅读“我的解答正确版本”，重点看自己的 SQL 应该怎么改。
4. 复盘“我的复盘要点”，把错因归类到题型总结中。
