# 产品年度销售额与竞品对比

## 1. 题目描述

题目来源：牛客 SQL117（中等），[深入分析各款产品年总销售额与竞品的年度对比](https://www.nowcoder.com/practice/99cc7f1798a84645a6aca5bdfd163fdb)。公开解法依据：[牛客题解](https://www.nowcoder.com/discuss/860309138532749312)。

OPPO 公司为了评估产品在 2023 年的市场表现，需要根据以下三个数据表，分析各款产品的全年销售总额及其与主要竞品全年销售总额的差值。

给定 `oppo_products_detail`（OPPO 产品详情）表，字段如下：

| 字段名 | 含义 |
|---|---|
| `product_id` | 产品 ID，产品的唯一标识符 |
| `product_name` | 产品名称 |
| `product_category` | 产品类别，固定为智能手机、平板电脑或智能手表 |

示例数据：

| product_id | product_name | product_category |
|---:|---|---|
| 1 | Find X7 | 智能手机 |
| 2 | Pad Pro 2 | 平板电脑 |

给定 `sales_info`（销售信息）表，字段如下：

| 字段名 | 含义 |
|---|---|
| `sale_id` | 销售 ID，销售记录的唯一标识符 |
| `product_id` | 产品 ID，用于关联产品详情表 |
| `sales_channel` | 销售渠道，固定为线上或线下 |
| `region` | 产品销售地区 |
| `quarter_1_sales_amount` | 该地区、该销售渠道的第一季度销售额 |
| `quarter_2_sales_amount` | 该地区、该销售渠道的第二季度销售额 |
| `quarter_3_sales_amount` | 该地区、该销售渠道的第三季度销售额 |
| `quarter_4_sales_amount` | 该地区、该销售渠道的第四季度销售额 |

示例数据：

| sale_id | product_id | sales_channel | region | quarter_1_sales_amount | quarter_2_sales_amount | quarter_3_sales_amount | quarter_4_sales_amount |
|---:|---:|---|---|---:|---:|---:|---:|
| 1 | 1 | 线上 | 华北 | 10000 | 15000 | 12000 | 18000 |
| 2 | 2 | 线下 | 华南 | 8000 | 10000 | 9000 | 13000 |
| 3 | 2 | 线下 | 华东 | 8000 | 10000 | 9000 | 13000 |
| 4 | 1 | 线下 | 华东 | 6000 | 20000 | 8000 | 10000 |

给定 `competitor_analysis`（竞品分析）表，字段如下：

| 字段名 | 含义 |
|---|---|
| `analysis_id` | 竞品分析 ID，竞品记录的唯一标识符 |
| `product_id` | 对应的 OPPO 产品 ID，用于关联产品详情表 |
| `competitor_name` | 竞品名称 |
| `total_competitor_sales_amount_2023` | 竞品 2023 年全年销售总金额 |

示例数据：

| analysis_id | product_id | competitor_name | total_competitor_sales_amount_2023 |
|---:|---:|---|---:|
| 1 | 1 | Apple iPhone 14 | 80000 |
| 2 | 2 | Huawei MatePad | 60000 |

要求查询每个产品 2023 年四个季度、全部地区和全部销售渠道的销售总金额，并计算该金额与主要竞品 2023 年全年销售总金额的差值。结果按照产品 ID 升序排列。

## 2. 期望结果

| product_id | product_name | competitor_name | total_sales_amount_of_product | sales_difference_with_competitor |
|---:|---|---|---:|---:|
| 1 | Find X7 | Apple iPhone 14 | 99000.00 | 19000.00 |
| 2 | Pad Pro 2 | Huawei MatePad | 80000.00 | 20000.00 |

## 3. 标准答案

```sql
WITH data AS (
    SELECT stack(
        2,
        1, 'Find X7', '智能手机',
        2, 'Pad Pro 2', '平板电脑'
    ) AS (product_id, product_name, product_category)
),
sales_info AS (
    SELECT stack(
        4,
        1, 1, '线上', '华北',
            CAST(10000 AS DECIMAL(10, 2)), CAST(15000 AS DECIMAL(10, 2)),
            CAST(12000 AS DECIMAL(10, 2)), CAST(18000 AS DECIMAL(10, 2)),
        2, 2, '线下', '华南',
            CAST(8000 AS DECIMAL(10, 2)), CAST(10000 AS DECIMAL(10, 2)),
            CAST(9000 AS DECIMAL(10, 2)), CAST(13000 AS DECIMAL(10, 2)),
        3, 2, '线下', '华东',
            CAST(8000 AS DECIMAL(10, 2)), CAST(10000 AS DECIMAL(10, 2)),
            CAST(9000 AS DECIMAL(10, 2)), CAST(13000 AS DECIMAL(10, 2)),
        4, 1, '线下', '华东',
            CAST(6000 AS DECIMAL(10, 2)), CAST(20000 AS DECIMAL(10, 2)),
            CAST(8000 AS DECIMAL(10, 2)), CAST(10000 AS DECIMAL(10, 2))
    ) AS (
        sale_id,
        product_id,
        sales_channel,
        region,
        quarter_1_sales_amount,
        quarter_2_sales_amount,
        quarter_3_sales_amount,
        quarter_4_sales_amount
    )
),
competitor_analysis AS (
    SELECT stack(
        2,
        1, 1, 'Apple iPhone 14', CAST(80000 AS DECIMAL(10, 2)),
        2, 2, 'Huawei MatePad', CAST(60000 AS DECIMAL(10, 2))
    ) AS (
        analysis_id,
        product_id,
        competitor_name,
        total_competitor_sales_amount_2023
    )
),
product_sales AS (
    SELECT
        product_id,
        CAST(
            SUM(
                quarter_1_sales_amount
                + quarter_2_sales_amount
                + quarter_3_sales_amount
                + quarter_4_sales_amount
            ) AS DECIMAL(20, 2)
        ) AS total_sales_amount_of_product
    FROM sales_info
    GROUP BY product_id
)
SELECT
    d.product_id,
    d.product_name,
    ca.competitor_name,
    ps.total_sales_amount_of_product,
    CAST(
        ps.total_sales_amount_of_product
        - ca.total_competitor_sales_amount_2023 AS DECIMAL(20, 2)
    ) AS sales_difference_with_competitor
FROM data d
INNER JOIN product_sales ps
    ON d.product_id = ps.product_id
INNER JOIN competitor_analysis ca
    ON d.product_id = ca.product_id
ORDER BY d.product_id;
```
