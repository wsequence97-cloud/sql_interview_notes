# 查询每门课都大于60分的学生成绩

## 1. 题目描述

有三张表：`student` 学生表、`class` 课程表、`sc` 选课成绩表，字段如下：

`student` 学生表：

| 字段名 | 含义 |
|---|---|
| `id` | 学生 id |
| `student_name` | 学生姓名 |

`class` 课程表：

| 字段名 | 含义 |
|---|---|
| `id` | 课程 id |
| `class_name` | 课程名称 |

`sc` 选课成绩表：

| 字段名 | 含义 |
|---|---|
| `sid` | 学生 id |
| `cid` | 课程 id |
| `score` | 课程成绩 |

示例数据：

`student`：

| id | student_name |
|---:|---|
| 1 | xing |
| 2 | long |
| 3 | hu |

`class`：

| id | class_name |
|---:|---|
| 1 | 语文 |
| 2 | 数学 |
| 3 | 英语 |

`sc`：

| sid | cid | score |
|---:|---:|---:|
| 1 | 1 | 61 |
| 1 | 2 | 62 |
| 1 | 3 | 63 |
| 2 | 1 | 59 |
| 2 | 2 | 71 |
| 3 | 2 | 71 |
| 3 | 3 | 72 |

要求查询：每一门已选课程成绩都大于 60 分的学生的所有科目成绩，输出学生姓名、课程名称、成绩。

> 本题口径：判断学生是否满足条件时，看该学生在 `sc` 表中已选课程的最低分是否 `> 60`。

## 2. 期望结果

| student_name | class_name | score |
|---|---|---:|
| xing | 语文 | 61 |
| xing | 数学 | 62 |
| xing | 英语 | 63 |
| hu | 数学 | 71 |
| hu | 英语 | 72 |

## 3. 标准答案

```sql
with student as (
    select stack(
        3,
        1, 'xing',
        2, 'long',
        3, 'hu'
    ) as (id, student_name)
),
class as (
    select stack(
        3,
        1, '语文',
        2, '数学',
        3, '英语'
    ) as (id, class_name)
),
sc as (
    select stack(
        7,
        1, 1, 61,
        1, 2, 62,
        1, 3, 63,
        2, 1, 59,
        2, 2, 71,
        3, 2, 71,
        3, 3, 72
    ) as (sid, cid, score)
),
pass_student as (
    select
        sid
    from sc
    group by sid
    having min(score) > 60
)
select
    s.student_name,
    c.class_name,
    sc.score
from student s
join pass_student p
    on s.id = p.sid
join sc
    on s.id = sc.sid
join class c
    on sc.cid = c.id
order by
    s.id,
    c.id;
```

## 4. 我的解答

```sql
WITH student_data AS (
SELECT 1 AS id, 'xing' AS student_name UNION ALL
SELECT 2, 'long' UNION ALL
SELECT 3, 'hu'
),
class_data AS (
SELECT 1 AS id, '语文' AS class_name UNION ALL
SELECT 2, '数学' UNION ALL
SELECT 3, '英语'
),
sc_data AS (
SELECT 1 AS sid, 1 AS cid, 61 AS score UNION ALL
SELECT 1, 2, 62 UNION ALL
SELECT 1, 3, 63 UNION ALL
SELECT 2, 1, 59 UNION ALL
SELECT 2, 2, 71 UNION ALL
SELECT 3, 2, 71 UNION ALL
SELECT 3, 3, 72
),
min_data as (
    select
        sid,
        min(score)
    from
        sc_data
    group by
        sid
    having min(score) > 60
)
    select
        s.student_name,
        c.class_name,
        sc.score
    from
        sc_data sc
    join min_data on sc.sid = min_data.sid
    join student_data s on sc.sid = s.id
    join class_data c on sc.cid = c.id
```

## 5. 我的解答正确版本

```sql
with student_data as (
    select 1 as id, 'xing' as student_name union all
    select 2, 'long' union all
    select 3, 'hu'
),
class_data as (
    select 1 as id, '语文' as class_name union all
    select 2, '数学' union all
    select 3, '英语'
),
sc_data as (
    select 1 as sid, 1 as cid, 61 as score union all
    select 1, 2, 62 union all
    select 1, 3, 63 union all
    select 2, 1, 59 union all
    select 2, 2, 71 union all
    select 3, 2, 71 union all
    select 3, 3, 72
),
min_data as (
    select
        sid,
        min(score) as min_score
    from sc_data
    group by sid
    having min(score) > 60
)
select
    s.student_name,
    c.class_name,
    sc.score
from sc_data sc
join min_data m
    on sc.sid = m.sid
join student_data s
    on sc.sid = s.id
join class_data c
    on sc.cid = c.id
order by
    s.id,
    c.id;
```

## 6. 我的复盘要点

- 【修正记法】这题要先按学生聚合，用 `having min(score) > 60` 找出“已选课程都大于 60”的学生，再回表取这些学生的所有成绩。
- 【错误原因】不能直接 `where score > 60` 后输出，否则 `long` 的数学 71 会被保留下来，但他还有语文 59，不满足“每门课都大于 60”。
- 【本题坑点】`min(score)` 只是判断学生是否合格的条件，最终输出仍然要回到成绩明细表 `sc`，不能只输出聚合结果。
