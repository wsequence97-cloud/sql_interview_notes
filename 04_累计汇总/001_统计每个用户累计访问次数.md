# 统计每个用户累计访问次数

## 1. 题目描述

给定表 `data`，字段如下：

| 字段名 | 含义 |
|---|---|
| `userid` | 用户 id |
| `visitdate` | 访问日期 |
| `visitcount` | 当次访问次数 |

示例数据：

| userid | visitdate | visitcount |
|---|---|---:|
| u01 | 2017/1/21 | 5 |
| u02 | 2017/1/23 | 6 |
| u03 | 2017/1/22 | 8 |
| u04 | 2017/1/20 | 3 |
| u01 | 2017/1/23 | 6 |
| u01 | 2017/2/21 | 8 |
| u02 | 2017/1/23 | 6 |
| u01 | 2017/2/22 | 4 |

要求查询：统计每个用户每个月的访问次数小计，并计算该用户截至当前月份的累计访问次数。

> 本题口径：同一用户同一月份的多条明细需要先求和；同一天出现多条记录也按多条访问明细累加，不做去重。

## 2. 期望结果

| 用户id | 月份 | 小计 | 累积 |
|---|---|---:|---:|
| u01 | 2017-01 | 11 | 11 |
| u01 | 2017-02 | 12 | 23 |
| u02 | 2017-01 | 12 | 12 |
| u03 | 2017-01 | 8 | 8 |
| u04 | 2017-01 | 3 | 3 |

## 3. 标准答案

```sql
with data as (
    select stack(
        8,
        'u01', '2017/1/21', 5,
        'u02', '2017/1/23', 6,
        'u03', '2017/1/22', 8,
        'u04', '2017/1/20', 3,
        'u01', '2017/1/23', 6,
        'u01', '2017/2/21', 8,
        'u02', '2017/1/23', 6,
        'u01', '2017/2/22', 4
    ) as (userid, visitdate, visitcount)
),
base as (
    select
        userid,
        from_unixtime(unix_timestamp(visitdate, 'yyyy/M/d'), 'yyyy-MM') as month,
        visitcount
    from data
),
monthly_data as (
    select
        userid,
        month,
        sum(visitcount) as month_cnt
    from base
    group by
        userid,
        month
)
select
    userid as `用户id`,
    month as `月份`,
    month_cnt as `小计`,
    sum(month_cnt) over(
        partition by userid
        order by month
        rows between unbounded preceding and current row
    ) as `累积`
from monthly_data
order by
    userid,
    month;
```

## 4. 我的解答

```sql
WITH data AS (
    SELECT 'u01' AS userid, '2017/1/21' AS visitdate, 5 AS visitcount UNION ALL
    SELECT 'u02' AS userid, '2017/1/23' AS visitdate, 6 AS visitcount UNION ALL
    SELECT 'u03' AS userid, '2017/1/22' AS visitdate, 8 AS visitcount UNION ALL
    SELECT 'u04' AS userid, '2017/1/20' AS visitdate, 3 AS visitcount UNION ALL
    SELECT 'u01' AS userid, '2017/1/23' AS visitdate, 6 AS visitcount UNION ALL
    SELECT 'u01' AS userid, '2017/2/21' AS visitdate, 8 AS visitcount UNION ALL
    SELECT 'u02' AS userid, '2017/1/23' AS visitdate, 6 AS visitcount UNION ALL
    SELECT 'u01' AS userid, '2017/2/22' AS visitdate, 4 AS visitcount
),
base as (
    select 
        userid,
        FROM_UNIXTIME(UNIX_TIMESTAMP(visitdate, 'yyyy/M/d'), 'yyyy-MM') month,
        visitcount
    from 
        data
),
mon as (
    select 
        userid,
        month,
        sum(visitcount) vis_cnt
    from 
        base
    group by userid,month
)
select 
    userid,
    month,
    vis_cnt,
    sum(vis_cnt) over(partition by userid order by month)
from 
    mon
```

## 5. 我的解答正确版本

```sql
with data as (
    select 'u01' as userid, '2017/1/21' as visitdate, 5 as visitcount union all
    select 'u02' as userid, '2017/1/23' as visitdate, 6 as visitcount union all
    select 'u03' as userid, '2017/1/22' as visitdate, 8 as visitcount union all
    select 'u04' as userid, '2017/1/20' as visitdate, 3 as visitcount union all
    select 'u01' as userid, '2017/1/23' as visitdate, 6 as visitcount union all
    select 'u01' as userid, '2017/2/21' as visitdate, 8 as visitcount union all
    select 'u02' as userid, '2017/1/23' as visitdate, 6 as visitcount union all
    select 'u01' as userid, '2017/2/22' as visitdate, 4 as visitcount
),
base as (
    select
        userid,
        from_unixtime(unix_timestamp(visitdate, 'yyyy/M/d'), 'yyyy-MM') as month,
        visitcount
    from data
),
mon as (
    select
        userid,
        month,
        sum(visitcount) as vis_cnt
    from base
    group by
        userid,
        month
)
select
    userid,
    month,
    vis_cnt,
    sum(vis_cnt) over(
        partition by userid
        order by month
        rows between unbounded preceding and current row
    ) as cum_cnt
from mon
order by
    userid,
    month;
```

## 6. 我的复盘要点

- 【我的卡点】这题我主要卡在日期字符串转月份：`2017/1/21` 这种格式要先按 `yyyy/M/d` 解析，再输出成 `yyyy-MM`。
- 【修正记法】斜杠日期只保留月份可以写成 `from_unixtime(unix_timestamp(visitdate, 'yyyy/M/d'), 'yyyy-MM')`。
- 【本题坑点】要先按 `userid + month` 汇总月度小计，再对月度结果累计，不能直接在原始明细上做窗口累计。
- 【本题坑点】`u02` 同一天有两条 `visitcount = 6`，期望小计是 12，说明本题不是去重用户访问，而是累加明细访问次数。
