# 统计直播间最大同时在线人数

## 1. 题目描述

给定直播间用户进出记录表 `data`，字段如下：

| 字段名 | 含义 |
|---|---|
| `room_id` | 直播间 id |
| `user_id` | 用户 id |
| `login_time` | 用户进入直播间时间 |
| `logout_time` | 用户退出直播间时间 |
| `dt` | 日期分区 |

示例数据：

| room_id | user_id | login_time | logout_time | dt |
|---|---|---|---|---|
| r01 | u01 | 20210310100000 | 20210310103000 | 20210310 |
| r01 | u02 | 20210310100500 | 20210310102000 | 20210310 |
| r01 | u03 | 20210310101000 | 20210310104000 | 20210310 |
| r01 | u04 | 20210310110000 | 20210310113000 | 20210310 |
| r02 | u05 | 20210310100000 | 20210310102000 | 20210310 |
| r02 | u06 | 20210310101000 | 20210310103000 | 20210310 |
| r02 | u07 | 20210310102500 | 20210310104000 | 20210310 |
| r03 | u08 | 20210310090000 | 20210310091000 | 20210310 |

要求查询：统计 `20210310` 当天每个直播间的最大同时在线人数。

> 本题口径：用户进入直播间记为在线人数 `+1`，用户退出直播间记为在线人数 `-1`，按时间顺序累计在线人数后取每个直播间的最大值。

## 2. 期望结果

| room_id | max_status |
|---|---:|
| r01 | 3 |
| r02 | 2 |
| r03 | 1 |

## 3. 标准答案

```sql
with data as (
    select stack(
        8,
        'r01', 'u01', '20210310100000', '20210310103000', '20210310',
        'r01', 'u02', '20210310100500', '20210310102000', '20210310',
        'r01', 'u03', '20210310101000', '20210310104000', '20210310',
        'r01', 'u04', '20210310110000', '20210310113000', '20210310',
        'r02', 'u05', '20210310100000', '20210310102000', '20210310',
        'r02', 'u06', '20210310101000', '20210310103000', '20210310',
        'r02', 'u07', '20210310102500', '20210310104000', '20210310',
        'r03', 'u08', '20210310090000', '20210310091000', '20210310'
    ) as (room_id, user_id, login_time, logout_time, dt)
),
event_data as (
    select
        room_id,
        user_id,
        login_time as event_time,
        1 as user_type
    from data
    where dt = '20210310'

    union all

    select
        room_id,
        user_id,
        logout_time as event_time,
        -1 as user_type
    from data
    where dt = '20210310'
),
online_status as (
    select
        room_id,
        user_id,
        event_time,
        user_type,
        sum(user_type) over(
            partition by room_id
            order by event_time, user_type
            rows between unbounded preceding and current row
        ) as status
    from event_data
)
select
    room_id,
    max(status) as max_status
from online_status
group by room_id
order by room_id;
```

## 6. 我的复盘要点

- 【我的卡点】这题的突破口是把 `login_time` 和 `logout_time` 拆成事件流，而不是直接拿原始区间做分组。
- 【修正记法】同时在线、区间重叠、库存变化这类题可以先想扫描线：开始时间 `+1`，结束时间 `-1`，按时间累计后取最大值。
- 【本题坑点】同一时间既有人退出又有人进入时，要明确先处理 `-1` 还是 `+1`，否则峰值在线人数可能差 1。
- 【下次先想】如果题目问的是“每小时最大在线人数”，就不能只按整天取最大，还要先确定小时粒度的统计口径。
